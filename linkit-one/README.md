# LinkIt One Code

Make sure your LinkIt One has the latest firmware on it.

## Circuit

![LinkIt One Board](https://raw.githubusercontent.com/idmontie/cool-humidor/master/linkit-one/board.jpg)

## Running

1. Open Arduino IDE
2. Open the .ino file
3. Connect your board
4. Compile and deploy to the board

If you connect to the Serial, you can see the debug statements. You should see
the board connect to WiFi and send humidity updates periodically.
